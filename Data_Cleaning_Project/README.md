# 📊 Data Cleaning & Reporting Automation Project

## 📌 Project Overview
This project automates data cleaning, analysis, and reporting using Python.  
It processes raw sales data and generates insights and visual reports.

---

## ⚙️ Technologies Used
- Python
- Pandas
- Matplotlib

---

## 📂 Project Structure
- cleaning.py → Data loading & cleaning
- analysis.py → KPI & business analysis
- visualization.py → Charts generation
- automation.py → Main pipeline
- raw_data/ → Input dataset
- charts/ → Generated graphs
- reports/ → Summary report

---

## 🔄 Workflow
1. Load dataset
2. Clean data (missing values, duplicates, formatting)
3. Perform analysis (sales KPIs, region/category trends)
4. Generate visualizations
5. Export report automatically

---

## 📊 Key Insights Generated
- Total Sales
- Total Orders
- Average Sales
- Top Performing Region
- Top Category
- Monthly Sales Trend

---

## 📈 Output
- Charts saved in `/charts`
- Report saved in `/reports/summary.txt`

---

This project demonstrates:
